Raspberry Pi Projects

Homemade Raspberry Pi Security camera

I was using a raspberry Pi 4 with the camera module attached to allow me to view my camera's perspective.

Here I installed a program for the raspberry pi operating system which is powered by OpenCV to detect objects in view of the camera. OpenCV is a programming library used for real-time computer vision, OpenCV comprises of object detection, face recognition and machine learning for applications to accelerate the use of machine perception in commerical products.

This was a very fun home made project to do in my spare time as I was able to freely experiment with how I want the camera to detect objects. For instance, I gave it an option to always send a email with a photo attachment every hour and left it infont of my garden window. It resulted in quite a lot of intresting photos of birds, insects and nature.

Homemade VPN Server

I Installed a raspberry Pi dependant VPN Server for my raspbery Pi for my own Wifi Network to create a secure Virtual Private Network. The software I installed for the Raspberry Pi is called PIVPN, this home-made bootstrap program is made purposely for the Rasberry Pi and is free open source server.

I tend to use PIVPN to for security purposes and internatioal puposes If I would like to access content around the world without having to pay for first part services such as NordVpn or ExpressVpn. It was very intresting to learn about how the fundementals of a VPN works and how it operates though out your whole Wifi-network.

Homemade AdBlocker

This was a fun homemade project to do. The convenience oh having my very own AdBlocker soley for my Wifi-network works like a dream. The software filters out HTTPS Ads thorugh your browsing activity on the internet.

This also applies for all devices connected to my Wifi-network, this made browsing the web more peaceful and run smoothly.

Retro Gaming Console

This was my favourite project to design. The raspberry Pi 4's computing power is strong enough to run emulators of you favourite retro consoles from Atari, SNES, Sega Genesis, N64 and so on.

What makes this project even more cool is the creativity I can use to make it unique. For instance I am able to SSH into the raspberry pi to add (Read Only-Memory) ROMs and customise the GUI for my Operating System. The free range to customise the Operating System.

The feasibility to run games of any retro console of your choice with cheats and custom configurations was fun to setup.
